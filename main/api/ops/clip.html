Clip¶

class nvtabular.ops.Clip(min_value=None, max_value=None, columns=None, preprocessing=True, replace=True)[source]¶

Bases: nvtabular.ops.TransformOperator

This operation clips values continous values so that they are with a min/max bound. For instance by setting the min value to 0, you can replace all negative values with 0. This is helpful in cases where you want to log normalize values.
